day20_api

API Project

공공 OpenAPI 호출, pagination, pandas 정리, folium 시각화를 통해 데이터 수집과 가공의 실전 감각을 쌓은 프로젝트.

day21_chatbot

Chatbot Project

텍스트 전처리, 패턴 매칭, NLP 기초를 연결해 대화형 프로젝트 구조를 직접 만져본 단계.

day23_attention_nmt

Attention NMT

RNN 기반 seq2seq에서 attention이 왜 필요한지, 번역 모델로 직접 체감한 프로젝트.

day24_Chroma_Fullstack

Chroma Fullstack

DINOv2 임베딩, ChromaDB 검색, Streamlit에서 FastAPI 풀스택 감각까지 확장된 대표 프로젝트.

day25_FridgeCook

FridgeCook

실사용에 가까운 아이디어를 Flask 웹앱으로 풀어내며 서비스화 감각을 키운 프로젝트.

visual_notes

Interactive Study Book

배운 내용을 다시 시각화 자산으로 정리해, 단순 복습을 넘어서 설명 가능한 형태로 바꾼 단계.
