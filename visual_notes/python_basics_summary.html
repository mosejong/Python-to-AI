핵심 개념 요약
Variables

변수와 자료형

Python은 값을 변수에 저장하고, 그 값을 조합하면서 프로그램 흐름을 만든다.

  • 정수, 실수, 문자열, 불리언 기본형
  • 리스트, 튜플, 딕셔너리, 세트 같은 컬렉션
  • 문자열 슬라이싱과 인덱싱은 자주 다시 보게 됨
Control Flow

조건문과 반복문

프로그램은 결국 “언제 실행하고, 얼마나 반복할지”를 정하는 구조다.

  • if / elif / else로 분기 처리
  • for문은 순회, while문은 조건 반복
  • break / continue 사용 위치 감각 중요
Function

함수

반복되는 코드를 묶어서 재사용하고, 입력과 출력을 분리하는 기본 도구다.

  • 매개변수와 반환값 개념
  • 작은 기능 단위로 쪼개는 습관
  • 프로젝트 코드가 길어질수록 함수 분리가 중요
def greet(name): return f"안녕하세요, {name}" print(greet("모세종"))
Collections

리스트와 딕셔너리

초반 Python 문제풀이에서 가장 많이 만나는 자료구조다.

  • 리스트: 순서 있는 데이터 묶음
  • 딕셔너리: key-value 매핑 구조
  • 반복문과 같이 쓰면 활용도가 급상승
OOP

클래스와 객체

데이터와 기능을 하나로 묶어서 관리하는 방식이다.

  • class로 설계, object로 사용
  • __init__으로 초기 상태 정의
  • 실습에서 bank_account 같은 구조가 대표적
class BankAccount: def __init__(self, owner, balance): self.owner = owner self.balance = balance
File I/O

파일 처리

Python이 실제 데이터와 연결되기 시작하는 지점이다.

  • open, read, write 기본
  • with 문으로 안전하게 다루기
  • 텍스트 파일 읽기/저장 감각은 나중에도 계속 연결됨
with open("note.txt", "w", encoding="utf-8") as f: f.write("오늘 복습 완료")
